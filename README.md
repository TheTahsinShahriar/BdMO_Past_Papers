## How to use this (Simple Guide)

### 1. Browse by Year
Click on any of the folder links in the **Quick Access** list above to see the questions from that specific year.

### 2. View or Download a Single File
1.  Inside a year's folder, click on the file you want (like `National_Secondary.pdf`).
2.  To **view** it: It will open directly in your browser.
3.  To **download** it: Click the **"Download"** button (looks like a downward arrow or says "Download") on the top right side of the file preview.

### 3. Download a Specific Year/Folder
GitHub doesn't easily allow downloading one folder at a time. The simplest way for beginners is:
*   Either download the **entire archive** (see below) and then keep only the folder you need.
*   Or open the folder and download the files one by one using the steps above.

### 4. Download Everything (The Whole Archive)
If you want to save **every single question paper** to your computer or phone at once:
1.  Go to the [main page of this repository](https://github.com/TheTahsinShahriar/BdMO_Past_Papers).
2.  Click the green **"<> Code"** button.
3.  Choose **"Download ZIP"**.
4.  Once downloaded, "Extract" the ZIP file to see all the folders.
